# MeetHalfWayUploadImg

### Launch project

```bash
git clone https://github.com/Axel-AMGHAR/MeetHalfWayUploadImg.git && cd MeetHalfWayUploadImg && cd back && npm install pm2 && npm install && cd ../front && npm install
```

Sur back et front
```bash
npm start 
```
