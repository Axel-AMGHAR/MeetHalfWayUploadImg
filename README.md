# MeetHalfWayUploadImg

Launch project

```bash
git clone https://github.com/Axel-AMGHAR/MeetHalfWayUploadImg.git && cd back && npm install && cd ../front && npm install
```

Sur back et front
```bash
npm start 
```

Lancer dans le navigateur 

/api/init_create

Avoir la photo de la ville 
/api/city_path/:code
